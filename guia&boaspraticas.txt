Para o python: 
foi criado um ambiente isolado com todos os comandos que vão ser necessários,
de forma a não interferir com o resto do computador.
- certificar que o python está instalado
- (no terminal) python -m venv venv: cria uma pasta à parte com bibliotecas, scripts, gitignore etc..
- (no terminal) .\venv\Scripts\Activate abrir a parta em questão e instalar as bibliotecas necessárias
- (no terminal) pip install pandas openpyxl para ler o ficheiro.
- para fazer dubug é sempre dentro do venv com python navios.py




como boa prática:
- também devemos de colocar as bibliotecas na parte inicial do projeto.
- comentar o projeto, para facilitar o acompanhamento
